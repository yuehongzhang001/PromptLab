导入 JSON 提示词

样本格式

{
  "sections": [
    {
      "title": "Role",
      "content": "You are a senior prompt engineer."
    },
    {
      "title": "Task",
      "content": "Rewrite the user's prompt into a cleaner production version."
    },
    {
      "title": "Constraints",
      "content": "Keep the intent unchanged.\nRemove redundancy.\nOutput only the final prompt."
    },
    {
      "title": "",
      "content": "Additional context block without title."
    }
  ]
}

支持字段：`sections`、`title`、`content`。`title` 可为空；所有区块都会按 `text` 处理。
